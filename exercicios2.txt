1) 
a) crie um repositório no github chamado exercicioAula03   
b) baixe ele para sua maquina (sugestão: c:\temp2\exercicio) (caso não tenha a pasta temp2 no c:\, crie a mesma)  
c) adicione 3 arquivos pdf e envie eles para o github  
d) com os arquivos no github, clone o repositório exercicioAula03 para uma outra pasta (sugestão c:\db1start\) (caso não tenha a pasta db1start no c:\, crie a mesma)  
e) remova 1 dos arquivos arquivos e envie para o github. Quando recarregar o github, deverão existir dois arquivos.  

2) utilizando o repositório exercicioAula03, realize:  
a) 3 commits independentes contendo somente arquivos do tipo html  
b) 2 commits independentes contendo somente arquivos do tipo docx  
c) 1 commit contendo somente arquivos xlsx  
d) 1 commit com remoção de pelo menos um arquivo html, um arquivo docx e um arquivo xlsx simultaneamente  
e) 1 commit com atualização de um arquivo html e outro arquivo xlsx  

o resultado final do exercício é que no seu repositório tenha  
* 2 arquivos html  
* 3 docx  
* 2 xlsx  
* 2 pdf (do exercicio 1)
